## 📚 Course Overview

This repository contains all materials related to the Deep Learning course for the 2nd semester of Computer Science, including:

- 📖 Course chapters and theory
- 🧪 Practical labs (TPs)
- 💻 Implementation in Python using Jupyter Notebooks
